# Prescribing Indicators

## PLDR

PLDR has summary datasets of the number and cost of medication prescribed from GP practices in England quarterly.

E.g. Prescribing indicators - Antidepressants (P_1_07) - https://pldr.org/dataset/prescribing-indicators-antidepressants-p107-emyye

https://pldr.org/datasets
