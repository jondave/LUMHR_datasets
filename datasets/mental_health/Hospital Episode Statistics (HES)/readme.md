# Hospital Episode Statistics (HES)

Private data: Contains highly sensitive patient-level details (diagnoses, surgeries, ages, and locations) for every admission, outpatient appointment, and A&E visit in NHS England hospitals. Access to this raw data is strictly controlled and only granted to approved researchers or healthcare planners via the Data Access Request Service (DARS).  

Public data: Anonymised and aggregated monthly and annual statistical reports.

Closed access.
