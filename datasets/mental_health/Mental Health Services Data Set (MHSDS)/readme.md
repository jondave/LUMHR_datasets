# Mental Health Services Data Set (MHSDS)

Private data: This is a patient-level database collecting clinical and operational records from individual children, young people, and adults interacting with mental health, learning disability, and autism services. Like HES, it is governed by strict UK GDPR laws and is not open to the public. 

Public data: Anonymized, high-level summaries and dashboards are published publicly to show how mental health services are performing across the country.  

Closed access.

https://digital.nhs.uk/data-and-information/data-collections-and-data-sets/data-sets/mental-health-services-data-set
